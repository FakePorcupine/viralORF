# Description

`viralORF` is an R package that finds all theoretically possible open reading frames (ORFs) in concatenated FASTA files.

# Installation instructions

```R
# loading devtools
if(!require(devtools)) { install.packages("devtools"); library(devtools) }

devtools::install_github("FakePorcupine/viralORF") # installs package
```

# Core functions

**ORF definitions** (Figure created with BioRender.com)
![ORF definitions](https://github.com/FakePorcupine/stop-stop-ORF-finder/blob/ec33dc054724a8f364b14fad8124fb438452532a/ORF%20definitions.png)

**1. `find_startstop_ORFs(file_name)`** 

Finds all start-stop ORFs, defined as a nucleotide sequence flanked by in-frame start (`"ATG"`) and stop codons, where overlaps are permitted.
* **Arguments**: `file_name` is file path to the input `.fasta` in your working directory.
* **Outputs**: Generates a spreadsheet mapping start-stop ORFs (`*_start-stop_ORFs.csv`), a canonical protein translation FASTA file (`*-translated.fasta`), and a txt file containing sequences excluded from analysis due to presence of ambiguous bases (`excluded_sequences.txt`).

**2. `find_stopstop_ORFs(file_name)`**

Finds all stop-stop ORFs, defined as a nucleotide sequence containing no in-frame stop codons.
* **Arguments**: `file_name` is file path to the input `.fasta` in your working directory.
* **Outputs**: Generates a spreadsheet mapping stop-stop ORFs (`*_stop-stop_ORFs.csv`), and a txt file containing sequences excluded from analysis due to presence of ambiguous bases (`excluded_sequences.txt`).

**3. `graph_startstop_ORFs(orf_data, output_prefix)`**

Generates publication-quality start-stop ORF visualizations using a clean 3x2 grid layout.
* **Arguments**: `orf_data` is either the active `data.frame` object returned by `find_startstop_ORFs()` or path `.csv` file generated by pipeline; `output_prefix` (optional) Character string giving prefix for output image files. Defaults to "ORF_plot".
* **Outputs**: Saves six individual frame-specific track plots (`*_start-stop_ORF.png`) and a consolidated A4 portrait composite panel laytout (`*_startstop_A4_composite_panel.png`) directly to your working directory.

**4. `graph_stopstop_ORFs(orf_data, output_prefix)`**

Generates publication-quality stop-stop ORF visualizations using a clean 3x2 grid layout.
* **Arguments**: `orf_data` is either the active `data.frame` object returned by `find_stopstop_ORFs()` or path `.csv` file generated by pipeline; `output_prefix` (optional) Character string giving prefix for output image files. Defaults to "ORF_plot".
* **Outputs**: Saves six individual frame-specific track plots (`*_stop-stop_ORF.png`) and a consolidated A4 portrait composite panel laytout (`*_stopstop_A4_composite_panel.png`) directly to your working directory.
